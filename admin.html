How to create a GitHub token
  1. Go to GitHub → Settings → Fine-grained tokens and click Generate new token
  2. Give it a name like maptap-admin and set expiration to 1 year
  3. Under Repository access, pick Only select repositories and choose your maptap repo
  4. Under Permissions → Repository permissions, set Contents to Read and write
  5. Click Generate token, copy it, and paste it in the GitHub connection card above
  6. The token is saved only in this browser's localStorage — never sent anywhere except GitHub's API directly
