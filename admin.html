How to update scores.json on GitHub
  1. Go to your GitHub repository and open scores.json
  2. Click the pencil icon (Edit this file) in the top-right
  3. Find the "scores": [ array and add the new entry (with a comma after the previous one)
  4. Scroll down and click Commit changes
  5. The leaderboard will update automatically within seconds
